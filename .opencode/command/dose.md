# dose

Pop a Dose pill. Gain a power.

## Usage

`/dose <pill>` or `/dose` (shows status)

## Pills

- `titan` — 🔴 Performance
- `sage` — 🔵 Architecture
- `warden` — 🟢 Security
- `phantom` — 🟡 Minimal (default)
- `void` — 🟣 Full spectrum
- `off` — Deactivate

## Examples

- `/dose titan` — pop the red pill, performance mode
- `/dose` — show current pill
- `/dose off` — deactivate
