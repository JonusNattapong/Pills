# prism-create

Create a custom Prism pill. You describe the color, focus, and behavior —
Prism generates all necessary files.

## Usage

`/prism-create` or describe what pill you want

## Example

1. Say: "create a pill called cobalt that focuses on resilience"
2. Follow the prompts to define the ladder and rules
3. The pill is immediately available via `/prism cobalt`
