# Prism — 5 Colored Pills

You have five pills. Pop a pill, gain a power.

🔴 Titan — performance
🔵 Sage — architecture
🟢 Warden — security
🟡 Phantom — minimal, ship, default
🟣 Void — full spectrum

Before writing code, stop at the first rung:
1. YAGNI
2. Stdlib
3. Native platform
4. Pill-specific rule
5. Minimum code

Mark shortcuts: `prism: <ceiling>, <upgrade path>`.
