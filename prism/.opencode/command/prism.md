# prism

Pop a Prism pill. Gain a power.

## Usage

`/prism <pill>` or `/prism` (shows status)

## Pills

- `titan` — 🔴 Performance
- `sage` — 🔵 Architecture
- `warden` — 🟢 Security
- `phantom` — 🟡 Minimal (default)
- `void` — 🟣 Full spectrum
- `off` — Deactivate

## Examples

- `/prism titan` — pop the red pill, performance mode
- `/prism` — show current pill
- `/prism off` — deactivate
